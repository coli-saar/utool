Provides basic classes for implementing codecs. A codec is a class which supports the conversion of an underspecified representation in some concrete syntax into a labelled dominance graph (input codec) or vice versa (output codec).

This package contains the following groups of classes:

  • the base classes {@link de.saar.chorus.domgraph.codec.InputCodec}, {@link de.saar.chorus.domgraph.codec.OutputCodec}, {@link de.saar.chorus.domgraph.codec.GraphOutputCodec}, and {@link de.saar.chorus.domgraph.codec.PluggingOutputCodec}, from which all concrete codec classes are derived;
  • classes for dealing with codec management: {@link de.saar.chorus.domgraph.codec.CodecManager};
  • classes for various exception that can occur while either using or registering a codec.
Codecs are classes with a fairly strictly defined interface. They are not intended to be instantiated by an application directly. Instead, they are registered in a codec manager object by their name and associated filename extension, and then the application can request fresh codec objects by name or extension from the codec manager.
