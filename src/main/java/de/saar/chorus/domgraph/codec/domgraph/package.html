Provides codecs for representing labelled dominance graphs in various graph representation languages. The following codecs are currently defined:
  • domgraph-gxl (input, output): dominance graphs in the GXL graph representation format;
  • domgraph-udraw (output): dominance graphs in the concrete syntax of the uDraw(Graph) graph drawing system.
  • domgraph-dot (output): dominance graphs in the Dot representation language.
